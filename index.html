Info Boxes

Pour des informations supplémentaires, il y a 2 styles d'info boxes :

  • Positive information : insérer un tableau d'une seule cellule avec une couleur de background = vert clair 3
  • Negative information : Idem mais avec une couleur = orange clair 3

Positive information : ex les bonnes pratiques ou les tips pour gagner du temps

Negative information : ex les avertissements et les restrictions d'API

Vidéos Youtube

Pour insérer une vidéo Youtube on peut :

  • Ajouter une image (ex une capture d'écran de la vidéo). Cette capture d'écran n'a pas d'importance car elle sera remplacée par la vidéo.
  • Ajouter un "Text alternatif" à l'image en faisant Cmd+Alt+Y ou Clique droit > "Text Alter.."
  • Insérer le lien dans la Description avec le format : https://www.youtube.com/watch?v=[video_ID]

Pour l'instant on ne peut pas paramétrer la date de début de la vidéo

Générer les fichiers

  1. Créer un dossier qui contiendra les fichiers générés
  2. Ouvrir un terminal et se positionner dans ce dossier
  3. Copier le DocId de votre google doc. C'est la longue chaîne de caractères vers la fin de l'URL (juste après le docs.google.com/document/d/)
  4. Dans le terminal taper la commande :
claat export <DocId>
